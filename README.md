# SDF Converter

## Features

- SVG and PNG support
- Saves settings to shareable URLs
- Bulk image conversion
- Runs on your GPU
- Customizeable
  - Spread
  - Alpha threshold
  - Output bias
  - Up res factor
  - Inside & outside colors
